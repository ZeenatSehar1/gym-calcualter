# Gym Fee Calculator

A simple Streamlit app to calculate gym membership fees.

## How to run locally

```bash
pip install streamlit
streamlit run gym-cal.py
```

## How to deploy on Streamlit Community Cloud
1. Push this repo to GitHub (already done).
2. Go to https://share.streamlit.io, sign in with GitHub, and click "New app".
3. Select the repository `ZeenatSehar1/gym-calcualter`, branch `main`, and the file `gym-cal.py`.
4. Click "Deploy".
